# Meteor Moon Phase

Shows an image with the current phase of the moon.

----

## Getting Started


```
# Install Meteor
curl https://install.meteor.com | /bin/sh

# Install Meteorite
sudo npm install -g meteorite

# Start the App (only need sudo the first time)
sudo mrt
```
